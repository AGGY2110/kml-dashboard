# KML Group — Marketing Intelligence Dashboard

Dashboard laporan tim marketing untuk 3 brand:
- **Ngaciin** (TikTok, Shopee, Meta)
- **Zona Nyam** (TikTok, Shopee, Meta)
- **Sentral Basreng** (TikTok, Shopee, Meta)

## Cara Jalankan

```bash
npm install
npm run dev
```

Buka: http://localhost:5173

## Deploy ke Vercel (Gratis)

1. Push ke GitHub
2. Login vercel.com → Import repo → Framework: Vite → Deploy

## Tambah Logo Brand

1. Login sebagai Admin
2. Buka **Pengaturan** di sidebar bawah
3. Upload logo ke imgbb.com → paste Direct Link
4. Klik Simpan
